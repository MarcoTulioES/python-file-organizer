# Python File Organizer

Script em Python que organiza automaticamente arquivos em pastas com base em suas extensões.

Este projeto foi desenvolvido com o objetivo de praticar automação de tarefas utilizando Python e manipulação de arquivos no sistema operacional.

---

## Funcionalidades

- Identificação automática de tipos de arquivos
- Criação automática de pastas
- Organização de arquivos por categoria
- Automação de tarefas repetitivas

---

## Tecnologias Utilizadas

- Python
- Biblioteca os
- Biblioteca shutil

---

## Estrutura do Projeto

```
python-file-organizer
│
├── organizer.py
└── README.md
```

---

## Como Executar

1. Clonar o repositório

```
git clone https://github.com/MarcoTulioES/python-file-organizer.git
```

2. Entrar na pasta do projeto

```
cd python-file-organizer
```

3. Executar o script

```
python organizer.py
```
